# CH6. 스택 (Stack) 3

## 6-4 계산기 프로그램 구현

### 문제 6-2
- 중위 표기법의 후위 표기법으로의 변환
  - 문제 1: 3 + 2 \* 4
  - 문제 2: 2 \* 4 + 3
  - 문제 3: 2 \* 1 + 3 / 2
- 문제 1 풀이
  - 1: \* 4
    - Stack: + <-top
    - 변환수식: 3, 2
  - 2: emtpy
    - Stack: +, \* <-top
    - 변환수식: 3, 2, 4
  - 최종 답: 3, 2, 4, \*, +
- 문제 3 풀이
  - 1: 2 \* 1 + 3 / 2
    - Stack: empty
    - 변환수식: empty
  - 2: + 3 / 2
    - Stack: \* <-top
    - 변환수식: 2, 1
  - 3: / 2
    - Stack: + <-top
    - 변환수식: 2, 1, \*, 3
  - 4: empty
    - Stack: +, / <-top
    - 변환수식: 2, 1, \*, 3, 2
  - 최종 답: 2, 1, \*, 3, 2, /, +

### 문제 6-3
- 중위 표기법의 후위 표기법으로의 변환
  - 문제 : (1 \* 2 + 3) / 4
- 문제 풀이
  - 1: (1 \* 2 + 3) / 4
    - Stack: (, \* <-top
    - 변환수식: 1, 2
  - 2: ) / 4
    - Stack: (, + <-top
    - 변환수식: 1, 2, \*, 3
  - 3: empty
    - Stack: empty
    - 변환수식: 1, 2, \*, 3, +, 4, /
  - 최종 답: 1, 2, \*, 3, +, 4, /
