# [10989] 수 정렬하기 3

문제
- 주어진 수를 오름차순으로 정렬하되, 카운팅이나 기수 정렬을 사용

정답
```python
def counting_sort(A, k):
    B = [-1] * len(A)
    C = [0] * (k + 1)
    for a in A:
        C[a] += 1
    for i in range(k):
        C[i+1] += C[i]
    for j in reversed(range(len(A))):
        B[C[A[j]] - 1] = A[j]
        C[A[j]] -= 1
    return B
a = []
MAX_NUM = 10000
for _ in range(int(input())):
    a.append(int(input()))
result = counting_sort(a, MAX_NUM)
for j in range(len(result)):
    print(result[j])
```
