# Baekjoon 알고리즘 문제풀기

## A+B / A-B

문제
- 두 정수 A와 B를 받고, 연산하여 출력하는 프로그램

정답
```python
a, b = map(int, input().split())
print(a+b)
print(a-b)
```

---

## 개

문제
- 아래 주어진 개를 출력하는 프로그램

- 파이썬에서 '\'를 출력하고자 할때는 '\\'로 입력해야 한다.
- 줄 개행이 포함된 출력을 원할경우 `print('''내용''')`을 이용한다.

```
{%raw}
|\_/|
|q p|   /}
( 0 )"""\
|"^"`    |
||_/=\\__|
{%endraw}
```

정답
```python
print('''|\_/|
|q p|   /}
( 0 )"""\\
|"^"`    |
||_/=\\\\__|''')
```
